# student_mantainer

## Get Started

  `git clone git@github.com:josnunezg/student_mantainer.git`

### Rails
  ```
    $ cd api/
    $ bundle install
    $ rake db:create
    $ rake db:migrate
    $ rails s -p 3001
  ```

### Frontend
  ```
    $ cd frontend/
    $ npm install // or yarn install
    $ npm start // or yarn start
  ```

## Test Api

To test api you can use [GraphiQL](https://electronjs.org/apps/graphiql)
